# **App Name**: StackStruct

## Core Features:

- Project Type Selection: Allows the user to select the type of project: 'Frontend + Backend', 'Frontend only', or 'Backend only'.
- Framework Selection: Lets the user choose from various frontend frameworks (React, Next.js, Vue, SvelteKit, Angular).
- Dynamic Options: Presents framework-specific options (e.g., TypeScript, React Router, state management).
- Dependency Management: Enables users to select styling (Tailwind CSS), utility libraries (Axios, React Query), form libraries, and add extra npm packages.
- Backend Configuration: Allows enabling/disabling a Node + Express backend, choosing between TypeScript/JavaScript, REST/basic auth routes, and setting a backend subfolder name.
- Backend Framework Selection: Allows the user to select from various backend frameworks (Express, NestJS, Fastify, Hapi, Koa).
- Folder Presets: Allows the user to select folder structure presets (Minimal/Standard/Advanced) for each framework.
- Monorepo Support: Supports monorepo structure with /frontend + /backend structure.
- Script Generation: Generates a Git Bash script to create folders, bootstrap the project, install dependencies, clean boilerplate, and create folder structures/starter files.
- Folder/File Structure Visualization: Shows a visual tree of the generated folder structure and file previews.

## Style Guidelines:

- Primary color: Ocean Blue (#464EB8), Medium Slate (#7B83EB), and Iris (#505AC9) from the Microsoft Teams branding.
- Background color (Light Mode): #F8F9FA.
- Surface color (Light Mode): #FFFFFF.
- Border color (Light Mode): #EDEEF0.
- Text color (Light Mode): #323130.
- Secondary text color (Light Mode): #605E5C.
- Accent color (Light Mode): #464EB8.
- Background color (Dark Mode): #252423.
- Surface color (Dark Mode): #323130.
- Border color (Dark Mode): #484644.
- Text color (Dark Mode): #F3F2F1.
- Secondary text color (Dark Mode): #D2D0CE.
- Accent color (Dark Mode): #7B83EB.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Responsive layout with left configuration panel and right preview panel, using rounded cards and smooth hover states.
- Use simple, outline-style icons from a library like Heroicons for visual clarity.
